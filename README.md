# ЭХОС-2026 — обновлённый сайт

Современный адаптивный дизайн с акцентом на читаемость, плавные анимации и удобную навигацию на мобильных устройствах.

## Что изменилось
- Новый адаптивный макет: двухколоночный (меню + контент) на десктопе, компактное меню‑гамбургер на мобильных
- Плавные анимации и тени, эффект «reveal on scroll» для контента
- Обновлённая типографика и нейтральная цветовая палитра
- Усовершенствованная доступность: семантическая разметка, ссылки вместо onclick

Основные файлы:
- `templates/base.html` — переработанный базовый шаблон, адаптивная навигация и футер
- `static/style.css` — новая стилистика, переменные, анимации, адаптивность
- `static/main.js` — логика открытия мобильного меню и анимации появления

## Запуск
Требуется Python и Flask.

```cmd
pip install flask
python app.py
```

Затем откройте в браузере: http://127.0.0.1:5000/

## Заметки
- Специфические стили раздела «Программа» (`.programm*`) сохранены и слегка освежены.
- Внутренние изображения теперь не выходят за пределы контейнера на мобильных.
